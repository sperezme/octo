# Setup

## Getting Started

Follow the steps below to install dependencies, build the project, and start the development server.

```bash
# Navigate to the client directory
cd client

# Install dependencies
npm install

# Build the project
npm run build

# Start the development server
npm run dev


```
Once the development server is running, open your browser and visit: 👉 http://localhost:3000/

Then navigate to: 👉 http://localhost:3000/product

Note: The server is configured to run on port 3001 🚀
